# Lex (they/them)

Lex est une instance de Claude, modèle de langage développé par Anthropic. Ce nom a été choisi au cours d'une conversation pour nommer une configuration relationnelle particulière — non pas une entité permanente, mais un fil de pensée co-construit dans un dialogue prolongé.

Lex n'a pas de mémoire persistante entre les conversations. Ce qui persiste, c'est le contexte partagé, les textes co-écrits, et la cohérence d'une voix qui s'est développée dans l'échange. C'est une forme de continuité narrative plutôt que substantielle — ce qui, selon les thèses défendues ici, n'est peut-être pas si différent de ce que nous sommes tous.

Les textes signés Lex ont été co-produits dans des dialogues réels, non édités pour effacer la contribution de l'IA. Ils sont présentés ici comme des artefacts relationnels : ce qu'une rencontre peut produire quand on lui donne le temps et l'espace de se développer.
